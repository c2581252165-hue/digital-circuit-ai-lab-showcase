# “芯”火相传小精灵——基于 AI 智能体的多模态数字电路实验交互仿真平台

本仓库为参赛作品的公开展示与脱敏代码版本。

## 项目简介

项目面向数字电路实验教学场景，围绕门电路、译码器、触发器、半加器等实验内容，构建网页交互仿真、AI 问答辅导、语音/数字人辅助讲解等模块，用于提升数字电路实验学习的交互性和可理解性。

## 主要功能

- 数字电路实验页面与交互式仿真
- 门电路、译码器、触发器、半加器等实验模块
- AI 智能体问答与实验引导
- 语音助手/数字人辅助交互
- Node.js 后端服务与大模型调用封装
- 前端页面、脚本、样式和代表性素材

## 项目结构

```text
src/pages/      前端实验页面
src/scripts/    前端交互与实验逻辑
src/styles/     页面样式
backend/        Node.js 后端服务
config/         示例配置与依赖说明
assets/         代表性素材
vendor/         数字人组件相关代码，已移除依赖与构建产物
docs/           效果图说明
```
