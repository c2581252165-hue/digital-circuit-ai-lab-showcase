# “芯”火相传小精灵——基于 AI 智能体的多模态数字电路实验交互仿真平台

“芯”火相传小精灵是面向数字电路实验教学的多模态交互仿真平台，整合网页实验仿真、AI 问答辅导、语音交互和数字人讲解等能力。

## 项目简介

项目围绕门电路、译码器、触发器、半加器等实验内容构建可交互的学习流程，使学生能够在实验页面中完成操作、观察反馈，并通过智能体辅助理解实验原理。

## 主要功能

- 数字电路实验页面与交互式仿真
- 门电路、译码器、触发器、半加器等实验模块
- AI 智能体问答与实验引导
- 语音助手/数字人辅助交互
- Node.js 后端服务与大模型调用封装
- 前端页面、脚本、样式和代表性素材

## 项目结构

```text
src/pages/      前端实验页面
src/scripts/    前端交互与实验逻辑
src/styles/     页面样式
backend/        Node.js 后端服务
config/         示例配置与依赖说明
assets/         代表性素材
vendor/         数字人组件相关代码
docs/           效果图说明
```
